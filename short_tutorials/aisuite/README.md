# Getting Started

- Let's get started by settings up your keys
    - https://console.groq.com/keys
    - https://platform.openai.com/

- create a python environment
```bash
python3 -m venv .venv
source .venv/bin/activate
pip3 install -r requirements.txt 
```

- run the code
```bash
python3 run.py 
```
